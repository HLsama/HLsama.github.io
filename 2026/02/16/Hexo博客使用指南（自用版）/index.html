Hexo博客使用指南（自用版）

2026/2/16 工具使用指南 hexo

为了防止每次想写博客都忘记怎么使用，这里直接写个博客来记录使用流程

从创建文章到上传：

首先来到博客的跟目录下，我的电脑是：新建卷D/ElysiaLove。然后右键空白处，点击Open Git Bash Here。如下图所示：

这样之后就可以创建一个命令框，我们使用各种指令来上传：

然后就可以开始创建文章了。

首先是创建文章的命令：hexo new “文章标题”。敲完回车就行，新建的文章会保存在_posts下，其中.md文件是文章本体，打开后即可编辑，同名文件夹是储存这个文章所需图片的地方。

写好文章就可以在命令框内敲：hexo g。它会生成静态的文章文件。

如果想要预览的话就接着：hexo s。他可以生成个链接，在本地预览这个文章。

如果没有问题的话就：hexo d。将其部署到Git上。
