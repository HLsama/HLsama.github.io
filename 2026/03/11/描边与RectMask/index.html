2.Soble算子实现描边

在讲卷积之前稍微提一下，若想要发光的描边的话，需要进行屏幕后处理，所以需要将Canvas的RenderMode改成Screen Space—Camera，并挂在对应的摄象机。

2.1卷积

在实现描边之前我们要先了解一个非常重要的定义：卷积。它其实本质就是加权平均，它会有个卷积核，是一个四方形的网格结构（例如3x3,4x4），然后卷积核内的每个像素都有一个权重值。当我们使用卷积核的时候，会将卷积核的中心放在该像素上，然后读取周围加权后的像素值并求和，得到新的像素值。

它虽然简单，但能实现很多图像处理效果，例如模糊、边缘检测等。卷积操作最重要的地方就是卷积核，它也被称作算子。我们可以想想边的概念是如何形成的：如果两个像素之间有明确的颜色、亮度等属性差异，我们就会认为他们之间存在界限，这就是描边了。而这样的不同像素之间的差异值我们可以用梯度来表示。基于这样的想法，出现了几种不同的边缘检测算子，而这次我们使用的就是Soble算子。

在片源着色器中，我们首先需要计算出像素的亮度，以亮度梯度为基准判断两个像素之间是否为边缘。根据人眼的感光细胞对RGB三种光的敏感度不同有这样的颜色—亮度转换公式：

float luminace(float3 color)
{
    return 0.2125*color.r+0.7154*color.g+0.0721*color.b;
}

至于卷积操作，我们只需要仿照式子带入代码即可：

float Sobel( sampler2D tex,float2 uv[9])
{
    const half Gx[9] ={-1,-2,-1,0,0,0,1,2,1};
    const half Gy[9] ={-1,0,1,-2,0,2,-1,0,1};
    float texcolor;
    float edgeX = 0;
    float edgey = 0;
    for(int it = 0;it<9;it++)
    {
        texcolor = luminace(tex2D(tex,uv[it]));
        edgeX += texcolor*Gx[it];
        edgey += texcolor*Gy[it];
    }
    float edge = 1-abs(edgeX)-abs(edgey);
    return edge;
}

这样我们就计算出了梯度值，也就是我们要的描边。将其乘上颜色，就能作为描边的颜色，将其输入到Alpha通道，即可得到镂空的描边效果。
