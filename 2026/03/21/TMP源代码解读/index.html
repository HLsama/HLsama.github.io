TMP源代码解读

2026/3/21 技术 UI

已经多次说过TMP最大的特殊点在于基于SDF的文字渲染，它可以几乎无损失的放大和缩小文字，而不产生锯齿，下文将结合TMP的源代码来介绍SDF，并对其中的延申内容进行说明。

unity版本：Unity6000.062f1

1.SDF简介

模型的表达方式有很多种，大体可以分成两派，本别是显式表达和隐式表达，显式表达比较常见，我们常用的3D模型，用顶点连成线，线构成面的方式就是显式的模型表达。而隐式的表达有点云、有向距离场（SDF）等方式。有向距离场就是这样的一种隐式模型表达方式。从命名就可以看出它的特点，有向代表他是有方向的，会用正负号来表达，而距离则说明了它表达的方式，具体来说，我们不直接描述几何的位置本身，而是描述空间中各点到模型的位置，模型外则是正值，内部则是正值。

SDF的表达方式就是通过距离找到几何的“表面”，所以它在求交上非常简单：

此外，在卡通渲染中现在最常见的方式之一就是使用SDF图来制作角色的面部阴影，这种方式可以完全自定义阴影的形态，并可以在不同形态之间有非常自然的过度：

1.1、基本几何

基于这种思想，我们可以通过一些式子来进行一些基础的几何表达：

圆：

 float circleSDF(float2 uv, float radius)
{
    float result = length(uv - float2(0.5,0.5)) - radius;
    return result;
}

在上面代码中，我们将UV当作空间中的各点，只要计算出uv到圆心的距离即可得到一个非常基础的圆形。

本文重点是关于文字这种复杂几何的SDF表达，所以基础几何就先不过多介绍了。

1.2、文字SDF

对于复杂的文字这种几何，我们就不可能实时的用式子来表达，只能将它预存到贴图中，在使用的时候实时读取。

1.2.1、SDF图的生成

至于生成这样贴图的方式基于以下的基本思想：对于任意我们想要制作成为SDF图的源图，我们将每个像素分为在几何里面和几何外面两部分，记为in和out两种，然后目标贴图的每个像素，找到源纹理中所对应的像素，并计算这个像素最近的，不同于自己属性的像素的距离，最后将所有结果Clamp到0~1之间。

下文是一个示例代码：

public static void GenerateSDF(Texture2D source, Texture2D destination)
{
    int sourceWidth = source.width;
    int sourceHeight = source.height;
    int targetWidth = destination.width;
    int targetHeight = destination.height;

    pixels = new Pixel[sourceWidth, sourceHeight];
    targetPixels = new Pixel[targetWidth, targetHeight];
    Debug.Log("sourceWidth" + sourceWidth);
    Debug.Log("sourceHeight" + sourceHeight);
    int x, y;
    Color targetColor = Color.white;
    for (y = 0; y < sourceWidth; y++)
    {
        for (x = 0; x < sourceHeight; x++)
        {
            pixels[x, y] = new Pixel();
            if (source.GetPixel(x, y) == Color.white)
                pixels[x, y].isIn = true;
            else
                pixels[x, y].isIn = false;
        }
    }


    int gapX = sourceWidth / targetWidth;
    int gapY = sourceHeight / targetHeight;
    int MAX_SEARCH_DIST = 512;
    int minx, maxx, miny, maxy;
    float max_distance = -MAX_SEARCH_DIST;
    float min_distance = MAX_SEARCH_DIST;

    for (x = 0; x < targetWidth; x++)
    {
        for (y = 0; y < targetHeight; y++)
        {
            targetPixels[x, y] = new Pixel();
            int sourceX = x * gapX;
            int sourceY = y * gapY;
            int min = MAX_SEARCH_DIST;
            minx = sourceX - MAX_SEARCH_DIST;
            if (minx < 0)
            {
                minx = 0;
            }
            miny = sourceY - MAX_SEARCH_DIST;
            if (miny < 0)
            {
                miny = 0;
            }
            maxx = sourceX + MAX_SEARCH_DIST;
            if (maxx > (int)sourceWidth)
            {
                maxx = sourceWidth;
            }
            maxy = sourceY + MAX_SEARCH_DIST;
            if (maxy > (int)sourceHeight)
            {
                maxy = sourceHeight;
            }
            int dx, dy, iy, ix, distance;
            bool sourceIsInside = pixels[sourceX, sourceY].isIn;
            if (sourceIsInside)
            {
                for (iy = miny; iy < maxy; iy++)
                {
                    dy = iy - sourceY;
                    dy *= dy;
                    for (ix = minx; ix < maxx; ix++)
                    {
                        bool targetIsInside = pixels[ix, iy].isIn;
                        if (targetIsInside)
                        {
                            continue;
                        }
                        dx = ix - sourceX;
                        distance = (int)Mathf.Sqrt(dx * dx + dy);
                        if (distance < min)
                        {
                            min = distance;
                        }
                    }
                }

                if (min > max_distance)
                {
                    max_distance = min;
                }
                targetPixels[x, y].distance = min;
            }
            else {
                for (iy = miny; iy < maxy; iy++)
                {
                    dy = iy - sourceY;
                    dy *= dy;
                    for (ix = minx; ix < maxx; ix++)
                    {
                        bool targetIsInside = pixels[ix, iy].isIn;
                        if (!targetIsInside)
                        {
                            continue;
                        }
                        dx = ix - sourceX;
                        distance = (int)Mathf.Sqrt(dx * dx + dy);
                        if (distance < min)
                        {
                            min = distance;
                        }
                    }
                }

                if (-min < min_distance)
                {
                    min_distance = -min;
                }
                targetPixels[x, y].distance = -min;
            }
        }
    }

    //EXPORT texture
    float clampDist = max_distance - min_distance;
    for (x = 0; x < targetWidth; x++)
    {
        for (y = 0; y < targetHeight; y++)
        {
            targetPixels[x, y].distance -= min_distance;
            float value = targetPixels[x, y].distance / clampDist;
            destination.SetPixel(x, y, new Color(1, 1, 1, value));
        }
    }
}

这个算法比较暴力，可以优化的点还很多，不过这个还不是本文的重点，TMP中也自带生成的工具，就不多介绍了。
